# Rapd
Super KISS PHP web app library.

## Quick start
```sh
mkdir myproject
cd myproject
echo "{}" >> composer.json
composer require asmundstavdahl/rapd:^0.5
cp -r vendor/asmundstavdahl/rapd/skeleton/* ./
php -S localhost:8080 --docroot=public/
```
